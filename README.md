# Hotel-Booking-Demand-Classification-Regression
This project implements a complete machine learning pipeline on the Hotel Booking Demand dataset (Kaggle), covering data cleaning, encoding, scaling, outlier treatment, and exploratory data analysis, followed by two predictive modeling tasks: classifying whether a booking will be cancelled (is_canceled) and predicting the Average Daily Rate (adr) of a booking.

# Features


Data cleaning (missing values, duplicates, target leakage removal)
Categorical encoding (label + one-hot)
Feature scaling and IQR-based outlier treatment
13-visualization exploratory data analysis
Multi-model classification (6 models: Logistic Regression, KNN, Decision Tree, Random Forest, Gradient Boosting, AdaBoost)
Multi-model regression (6 models: Linear Regression, KNN, Decision Tree, Random Forest, Gradient Boosting, AdaBoost)
